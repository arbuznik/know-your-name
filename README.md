# Know your name

### About

Make gender, age and nationality assumptions based on the given name.

Data is provided by public APIs:
https://genderize.io
https://agify.io
https://nationalize.io

### Stack
JS (ES6), CSS3, HTML5, OOP, async api requests, error handling.
